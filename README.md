# 🏠 Apartment Rent Prediction App

This Streamlit app predicts monthly apartment rent based on:
- Area (sq. ft.)
- Number of Bedrooms
- Number of Bathrooms

## 🔧 How to run:

1. Clone the repository  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run the app: `streamlit run app.py`

## 👩‍💻 Built using:
- Streamlit
- scikit-learn
- pandas, numpy
